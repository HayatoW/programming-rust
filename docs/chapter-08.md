推移依存クレート: 依存クレートの依存クレート

ライブラリ: main がない (rlib ファイルが作られる)

バイナリ: main がある。実行可能。

以下を多用していきたい

![make-module](./make-module.png)

一般的には、タイプやトレイトやモジュールをインポートし、関数や定数やその他の要素には相対パスでアクセスするのが良いスタイルであるとされる。

```rust
use std::mem; // モジュールのインポート

if s1 > s2 {
    // 関数は相対パスでアクセス
    mem::swap(&mut s1, &mut s2);
}
```
