{:?} フォーマット指定子で表示すると、デバッグ表示となる。

- err.source()
  - err の原因となったエラーがもしあれば Option 型で返す。

標準ライブラリのエラー型はスタックトレースを含まないが、スタックトレースを表示してくれるエラーを提供する anyhow クレートがあり、広く使われている。

標準ライブラリのエラー型はすべて以下に変換できる (すべてのエラーを表す)

```rust
Box<
    dyn std::error::Error + // 任意のエラーを表す
    // スレッド間の受け渡しが可能になっている
    Send +
    Sync +
    'static
>
```
